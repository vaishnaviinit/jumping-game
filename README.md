
# Platform Jumper

Platform Jumper is a web game built using **HTML**, **Tailwind CSS**, and **JavaScript**. Players jump on colorful platforms, avoid red bombs, and climb higher. The game features smooth physics, collision detection, score tracking, camera follow, and supports keyboard and mobile controls for an engaging, fun experience.

## Features

- **Smooth physics** and collision detection for realistic movement
- **Score tracking** to encourage competitive play
- **Camera follow** for dynamic, side-scrolling gameplay
- **Responsive controls** for keyboard and mobile
- Colorful platforms and challenging hazards (like red bombs)

## How to Play

- Use arrow keys (or touch) to move and jump
- Avoid red bombs; touching one ends the game
- Climb higher to increase your score
